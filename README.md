# 💆 O3 Spa and Salon

A premium, production-ready spa and salon website built using **React + Vite** for **O3 Spa and Salon**, located in **Whitefield, Bengaluru**.

The website is designed with a modern luxury UI, responsive layout, reusable business configuration, SEO optimization, WhatsApp booking integration, Google Maps support, and premium user experience.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/mukeshkumarsingh7488-afk/O3-Spa-And-Salon.git
```

Go inside the project

```bash
cd O3-Spa-And-Salon
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
